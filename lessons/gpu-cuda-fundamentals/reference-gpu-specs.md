## Quick Decision Guide

```
┌─────────────────────────────────────────────────────────┐
│ Task                        │ Recommended Hardware      │
├─────────────────────────────┼───────────────────────────┤
│ Learning / experimenting    │ Google Colab (free T4)    │
│ Fine-tuning 7B (QLoRA)      │ RTX 4090 or A100 40GB    │
│ Fine-tuning 13B (QLoRA)     │ A100 80GB                │
│ Training 7B from scratch    │ 4-8× A100 80GB           │
│ Training 70B from scratch   │ 64-256× H100             │
│ Training 175B+ from scratch │ 1000+ H100 or TPU v5p pod│
│ Inference (local, ≤13B)     │ Apple M3/M4 Max or 4090  │
│ Inference (serving at scale)│ H100, A100, or TPU v5e   │
│ Budget training             │ MI300X or Gaudi 2        │
└─────────────────────────────┴───────────────────────────┘
```
