# Go Track Roadmap

Welcome to the Go track. If you know Rust, you'll find Go refreshingly
simple — fewer knobs, faster compilation, and a language built for teams.

Go powers the cloud: Kubernetes, Docker, Terraform, Prometheus, etcd.
By the end of this track, you'll understand why.
